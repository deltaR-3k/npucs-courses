# 图灵班学习指南

<div style="text-align: center; font-family: 'Noto Serif SC'; font-size: 1.5em; font-weight: 600;" markdown="1">

折大之初，谁传道之？  
图灵四世，何由考之？  
思政军体，何以混之？  
通识选修，水之乐之。  
数理基础，可求活之？  
专业课程，纵狂行之。  
远方何在，书评阅之；  
几代传承，一站括之。  
见此文者，愿能惜之；  
前人潸然，习坎避之。  
世世得进，恒者唯知！  

</div>
